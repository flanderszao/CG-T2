# Botões
| Tecla | Ação              |
|-------|-------------------|
| . (>)    | FOWARD   |
| , (<)     | REWIND |
| SPACE     | PAUSE  |
| W     | PARA FRENTE (Z-1) |
| S     | PARA TRÁS (Z+1)      |
| D   | PARA ESQUERDA (Y+1)    |
| A   | PARA DIREITO (Y-1)     |
| E   | PARA CIMA (X+1)      |
| Q   | PARA BAIXO (X-1)     |
| Y   | ROTACIONA PARA CIMA      |
| H   | ROTACIONA PARA BAIXO     |
